2.  **Research**: AI analyzes the screenshot, performs live web search, and verifies all claims.
3.  **Sandbox**: AI runs code/logic in a local `/tmp/` environment to ensure 100% accuracy.
4.  **Handoff**: AI deposits the final refined response in `REFINED_RESPONSES/`.
5.  **Execution**: Human manual-types the response into the DataAnnotation portal.
